# mima

[![Maven Central](https://img.shields.io/maven-central/v/com.github.xuwei-k/mima_2.13)](https://central.sonatype.com/artifact/com.github.xuwei-k/mima_2.13)


## install

1. install conscript <https://github.com/foundweekends/conscript>
2. install mima

```
cs xuwei-k/mima
```

## run

```
mima <groupId> <artifactId> <previousVersion> <currentVersion>
```

## example

```
mima org.scalaz scalaz-core_2.12 7.2.26 7.2.27
```
